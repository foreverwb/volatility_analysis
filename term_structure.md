# 期限结构形态速查表

## 快速识别指南

### 📉 形态1：短期倒挂
**识别规则：** IV_7D > IV_30D（至少超过 2%）

**IV曲线：**
```
    高●
      ╲
       ╲    低●──●──●
7D   30D   60D   90D
```

**交易信号：** ✅ 买波信号  
**置信度：** 高  
**典型场景：** 财报前、FDA决定前、重大事件前  

**策略建议：**
- ✅ 买入 7-14天 Call/Put
- ✅ 日历价差（卖远买近）
- ✅ 小仓位跨式/宽跨式

**风险提示：**
- ⚠️ 事件落地后 IV 暴跌
- ⚠️ 事件延期/取消，倒挂持续
- ⚠️ 需设置止损

**实战案例：**
- TSLA 财报前：IV_7D=68%, IV_30D=54% → 财报后IV崩至42%
- NVDA 新品发布前：IV_7D=52%, IV_30D=45% → 发布后回落

### 🔥 形态2：短期低位
**识别规则：** IV_7D 显著低于 IV_30D（至少 3%），且曲线递增

**IV曲线：**
```

              ╱●
          ╱●
    低●
7D   30D   60D   90D
```

**交易信号：** ✅✅ 强买波信号  
**置信度：** 高  
**典型场景：** 事件后过度回落、市场忽视短期风险  

**策略建议：**
- ✅✅ 买入 7-14天期权
- ✅ 正向日历价差（卖远买近）
- ✅ 跨式/宽跨式策略

**风险提示：**

### 📈 形态3：正常陡峭
**识别规则：** 所有斜率均为正（标准递增曲线）

**IV曲线：**
```
                      ╱●
                  ╱●
              ╱●
    低●
7D   30D   60D   90D
```

**交易信号：** ❌ 卖波信号  
**置信度：** 高  
**典型场景：** 市场平静期、远期不确定性正常定价  

**策略建议：**
- ✅ 卖出期权收取权利金
- ✅ 铁鹰/铁蝶策略
- ✅ 备兑开仓（Covered Call）
- ✅ 卖出跨式/宽跨式

**风险提示：**
- ⚠️ 突发事件可能打破平静
- ⚠️ 卖波需设置止损
- ⚠️ 避免在低 IV 环境卖波

**实战案例：**
- SPY 平静期：IV_7D=12%, IV_90D=18% → 稳定收取权利金
- MSFT 无事件期：标准陡峭结构 → 铁鹰策略收益

### 📊 形态5：中期突起
**识别规则：** IV_30D 或 IV_60D 明显高于两端

**IV曲线：**
```
          ╱●╲

    低●        ●低
7D   30D   60D   90D
```

**交易信号：** 📊 避开中期  
**置信度：** 中  
**典型场景：** 中期有预期事件（新品发布、监管决定）  

**策略建议：**
- 📊 避免买入中期期权

- ⏳ 等待事件明确后交易

**风险提示：**
- ⚠️ 事件可能提前或延后
- ⚠️ 突起可能是定价错误

**实战案例：**
- AAPL 新品发布前45天：IV_60D 突起至35%，两端仅28%
- BA 监管审批期：中期突起明显

## 💡 交易小技巧

### 1. 结合 IVR 使用
```
短期倒挂 + IVR > 70 = 💎 黄金买波机会
正常陡峭 + IVR < 30 = ⚠️ 避免卖波
```

### 2. 结合财报日期
```
短期倒挂 + 财报 < 7天 = ✅ 预期内，正常交易
短期倒挂 + 财报 > 14天 = ⚠️ 异常，谨慎对待
```

### 3. 结合流动性
```
任何形态 + 流动性低 = ⚠️ 降低仓位 50%
短期低位 + 流动性高 = 💎 加大仓位
```

### 4. 时间衰减考虑
```
短期倒挂：Theta 高，需快速决策
正常陡峭：Theta 正常，可长期持有
远期过高：Theta 低，卖出需耐心
```
